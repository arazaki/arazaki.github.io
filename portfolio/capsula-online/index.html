Cápsula do Tempo Online

01/2023 - current

Cápsula do Tempo Online is like an time Capsule web and mobile application that allows guests to record video messages during wedding events to be sent to the hosts (newlyweds) in the future.

Main Activities

  • ● Developed web application using ReactJS, HTML, and CSS
  • ● Created REST API using NodeJS, Cloud Functions, and Firebase
  • ● Developed the app using Flutter
  • ● Built a synchronization service
  • ● Implemented the CI/CD pipeline using Github Actions
  • ● Implemented the marketing website using GatsbyJS
